# Hexo部署命令

## 本地预览
hexo cl; hexo s
# 清理缓存并启动本地服务器预览

## 生成并部署
hexo cl; hexo g; hexo d
# 清理→生成→部署到远程服务器

## 分步执行
hexo clean      # 清理缓存
hexo generate   # 生成静态文件
hexo server     # 本地预览
hexo deploy     # 部署到远程


# 标准Git推送流程（推荐）

## 1. 添加所有修改的文件

## 3. 推送到GitHub
git push origin main
