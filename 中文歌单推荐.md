## 🔧 **如何更换歌单**

1. 打开网易云音乐网页版
2. 搜索您喜欢的歌单
3. 复制歌单ID（URL中的数字）
4. 修改 `themes/solitude/_config.yml` 中的 `id` 值

### 示例配置
```yaml
capsule:
  enable: true
  id: 3778678  # 更换这里的数字
  server: netease
  type: playlist
  volume: 0.6
```
